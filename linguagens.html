Principais Linguagens de programação!

Java é uma linguagem de programação de alto nível e orientada a objetos, famosa por sua portabilidade.
Graças à JVM (Java Virtual Machine), o código é compilado para um formato intermediário (bytecode) e;
pode ser executado em qualquer sistema operacional sem precisar ser reescrito


O Python é uma linguagem de programação de alto nível, interpretada e multiponto, conhecida por sua sintaxe clara e altamente legível.
É amplamente utilizada em inteligência artificial, análise de dados, automação e desenvolvimento web devido à sua vasta biblioteca de recursos


C++ é uma linguagem de programação compilada e de propósito geral. Criada como uma extensão da linguagem C, ela destaca-se pela alta performance,
eficiência e controle direto sobre o hardware. Suporta múltiplos paradigmas, como programação procedural, genérica e orientada a objetos.
